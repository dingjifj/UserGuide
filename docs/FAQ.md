# 常见问题

本章节收集了用户在使用过程中常见的疑问与解答，涵盖连接问题、速度问题、会员与订阅相关问题等。我们会持续更新这个章节，欢迎反馈更多问题与建议！

## 使用提示

* 体验受**本地网络质量、运营商策略、节点负载、目标应用自身状态**等影响。 
* 排查顺序建议：**检查网络 → 重新连接 → 切换节点/模式 → 重启应用/设备 → 更新客户端**。
* 若问题仍存在，请在联系客服时提供：**账号、设备型号、系统版本、客户端版本、所在国家/地区、当前网络（Wi-Fi/蜂窝）、报错截图/时间点**。

## A. 连接与登录

### Q1：点击“一键连接”一直转圈/连接失败怎么办？

**可能原因**：网络不可用、当前节点拥堵、系统 VPN 权限未授予、公司/校园网限制。

**解决方案**：

1. 检查网络连接，切换 Wi-Fi/蜂窝网络重试。
2. 关闭后重新打开应用，再点“一键连接”。
3. 更换节点/模式（尤其在高峰时段）。
4. 检查系统 VPN 权限设置，确保已授权。
5. 在公司/校园网环境下，尝试切换到其他网络（如手机热点）测试是否网络限制导致。

### Q2：提示“需要添加 VPN 配置/权限”，我该怎么做？

这是系统提示，用于建立安全连接。

**处理**：在弹窗中选择【允许/添加配置】并按系统要求验证（可能需要解锁/指纹/密码）。

### Q3：显示已连接，但国内 App 仍打不开/无变化？

**可能原因**：节点不适合当前场景、目标服务自身异常、App 有账号/地区限制、本地 DNS/代理冲突。

**解决方法**：

1. 切换到【智能选线】节点，或尝试其他节点。
2. 检查目标服务是否有公告/维护，或尝试访问其他国内服务验证是否加速生效。
3. 某些服务可能有账号/地区限制，尝试切换账号或联系服务商确认。
4. 检查本地是否有其他 VPN/代理软件冲突，尝试关闭后重试。

### Q4：为什么会频繁掉线/自动断开？

**可能原因**：网络不稳定、节点负载过高、系统资源限制、VPN 权限被系统收回。

**解决方案**：

1. 检查网络质量，切换网络重试。
2. 尝试更换节点/模式，避开高峰时段。
3. 在系统设置中允许应用后台运行（尤其 Android），避免系统资源限制导致断开。
4. 检查设备系统资源使用情况，关闭不必要的后台应用。
5. 确保系统 VPN 权限未被误操作收回，必要时重新授权。

## B. 速度慢 / 卡顿 / 延迟高

### Q1：速度慢、加载卡、延迟高怎么优化？

**可能原因**：当前节点不适合当前网络状况、目标服务服务器负载高、网络质量不佳、设备性能限制。

**解决方案**：

1. 切换到【智能选线】节点，或尝试其他节点。
2. 检查本地网络质量，切换 Wi-Fi/蜂窝网络重试。
3. 在高峰时段，节点负载可能较高，尝试避开高峰时段使用。
4. 切换节点：优先选择地理位置更近/负载更低的节点
5. 断开重连，或重启应用

### Q2：白天好用，晚上高峰就卡怎么办？

**可能原因**：高峰时段节点负载过高，导致速度慢、连接不稳定。

**解决方案**：

1. 尝试切换到【智能选线】节点，系统会自动选择当前最优节点。
2. 避开高峰时段使用，或在高峰时段尝试不同节点。
3. 如果高峰时段经常使用，建议升级套餐以获得更多节点/更高优先级。

### Q3：游戏延迟忽高忽低怎么办？

**可能原因**：网络抖动、节点距离不合适、游戏服务器自身波动。

**解决方案**：

1. 切换到【低延迟优先】模式，优先选择延迟更低的节点。
2. 尝试不同节点，找到适合游戏的节点。
3. 检查本地网络质量，切换 Wi-Fi/蜂窝网络重试。
4. 避免多人共享同一网络占用带宽。

## C. 账号与多设备管理

### Q1：提示“设备数量已满/达到上限”怎么办？

**原因**：你的套餐允许的同时在线设备数量已达到上限。

**解决方案**：

1. 进入【设备管理】，下线不常用设备。
2. 升级套餐以获得更多同时在线设备数。

### Q2：怀疑账号被他人使用，怎么办？

**解决方案**：

1. 立即进入【设备管理】查看已登录设备，远程下线不认识的设备。
2. 尽快修改账号密码，或绑定手机号/邮箱增加安全性。
3. 如果问题严重，建议联系客服寻求进一步帮助。

### Q3：换手机/换电脑后怎么迁移会员？

**解决方案**：

1. 在旧设备上登录账号，进入【设备管理】下线旧设备。
2. 在新设备安装应用并登录同一账号，完成登录后继续使用。

> 注意：跨平台迁移可能需要重新授权 VPN 权限。

## D. 订阅与支付

### Q1：如何取消订阅？

* **iOS**：系统【设置】→ Apple ID →【订阅】→ 选择本产品 →【取消订阅】
* Google Play：Play 商店 →【付款和订阅】→【订阅】→ 取消订阅
* 官网购买：登录官网账户 →【订单/订阅管理】→ 关闭自动

### Q2：为什么支付后会员没有生效？

**解决方法**：

1. 在【账号】页面点击【恢复购买】。
2. 退出账号重新登录。
3. 检查是否登录了购买时使用的同一账号。
4. 如果以上步骤无效，请联系在线客服提供订单信息（订单号/支付截图/购买账号等）以便核实处理。

### Q3：如何申请退款？

* **iOS/Google Play 内购**：通常需按平台规则在对应平台发起退款申请，等待审核结果。
* **应用内直购/官网订单**：通过应用内【我的 → 帮助与客服 → 退款申请】提交订单信息与退款理由，客服会在 2 个工作日内回复处理结果。

### Q4：为什么会被拒绝退款？

**常见原因**：

* 已成功开通且使用过会员权益。
* 个人原因（如不喜欢/不需要了等）。
* 违反服务条款或存在滥用行为导致的限制。
* 明显使用痕迹（例如已使用超过【1 小时/天】或消耗大量流量/节点资源）。

## E. 其他问题

### Q1：退出登录会影响会员吗？

退出登录不会影响会员资格，但需要重新登录才能继续使用会员权益。请确保记住账号信息以便重新登录。

### Q2：同一账号能不能手机和电脑同时用？

可以，但受套餐“同时在线设备数/同时连接数”限制。请根据你的套餐选择合适的设备数量使用，避免超过上限导致无法连接。

### Q3：连接时提示“无法创建 VPN 配置/配置失败”怎么办？

**可能原因**：系统 VPN 配置冲突、权限问题、设备限制。

**解决方案**：

1. 重启设备后重试。
2. 检查系统 VPN 权限设置，确保已授权。
3. 检查是否有其他 VPN/代理软件冲突，尝试关闭后重试。
4. 如果问题持续，请联系在线客服提供设备型号、系统版本、报错截图等信息以便排查。

### Q4：后台切出去就断开怎么办？

**解决方案**：

1. 在系统电池设置中关闭对本应用的省电限制。
2. 允许后台运行/自启动（不同机型路径不同）。
