# 认识归途

## 1. 产品概述

**产品名称**：归途回国加速 VPN

**产品介绍**：为海外用户提供稳定、低延迟的回国网络连接与内容访问体验，提升访问国内应用/网站的速度与成功率。

**产品定位**：跨境网络加速与连接服务（面向海外华人/出境人员/跨境工作者）。

## 2. 适用人群

* **海外华人/留学生**：需要流畅使用国内影音、社交、生活服务。
* **出差/旅行用户**：短期在海外希望保持国内服务可用性与速度。
* **跨境办公人群**：频繁使用国内协作工具、企业系统、网盘、会议等。
* **家庭用户**：多设备、多人共享，追求简单易用与稳定性。

## 3. 使用场景

* **国内影音娱乐**：国内视频/音乐/直播更流畅。
* **社交与通讯**：更稳定访问国内社交平台、图片/语音消息加载更顺畅。
* **支付与生活服务**：更顺畅使用国内电商、出行、外卖、银行/支付相关服务。
* **远程办公**：访问国内企业内网/业务系统更稳定。
* **游戏加速**：改善回国线路延迟与抖动。

## 4. 核心功能

### 4.1 一键连接

* 轻触即可完成连接，无需复杂设置。
* 新手也能快速上手。

### 4.2 智能选线 / 自动切换

* 根据你的位置、网络状况与节点负载自动选择最优线路。
* 线路波动时自动切换，减少中断。

### 4.3 多节点覆盖

* 覆盖多个地区节点。
* 支持按场景选择：影音优先 / 稳定优先 / 低延迟优先。

### 4.4 多设备支持

* 支持常见平台：iOS / Android / Windows / macOS。
* 同账号可在多设备使用。
